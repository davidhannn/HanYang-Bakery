# HanYang Bakery

## Description

An Ecommerce site that I created with GatsbyJs and Contentful as the Headless CMS. 

Demo: https://hanyangbakery.netlify.app/

#### Technologies

- GatsbyJs
- GraphQL
- SnipCart
- FormSpree


#### How to Setup and Run 

```html
  // Local build at http://localhost:8000/
  gatsby develop

  // Production build
  gatsby build
```
